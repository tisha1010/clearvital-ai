# ClearVital AI

AI-powered medical report analyzer that converts complex health data into simple, understandable insights.

---

## 🚀 Features

- Analyze medical markers (Hemoglobin, etc.)
- AI-generated explanations
- History tracking (MongoDB)
- Modern futuristic UI
- Dark / Light mode
- Full-stack architecture (MERN + Python)

---

## 🛠 Tech Stack

- Frontend: React (Vite + Tailwind)
- Backend: Node.js + Express
- Database: MongoDB
- AI: OpenRouter (Mistral / LLaMA)
- Python FastAPI (AI service)
